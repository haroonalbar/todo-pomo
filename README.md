# Pomodoro Timer

A macOS menu bar Pomodoro timer with integrated task management.

## Features

- **Timer** - Configurable work/rest durations with automatic transitions
- **Tasks** - Add, edit, reorder (drag & drop), and set deadlines
- **Session Linking** - Link tasks to active work sessions
- **Statistics** - Track completed sessions and work time
- **Notifications** - Sound alerts and system notifications
- **Persistence** - Settings, tasks, and stats saved automatically

## Build & Run

```bash
xcodebuild -scheme Pomodoro build && open -a Pomodoro
```
